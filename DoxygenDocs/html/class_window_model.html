Detailed Description

WindowModel class inherits RenderWindow so that more functionalities and interfaces can be added to each window objects. Some of the main functionalities of this class are: holding the renference to objects that can be rendered, rendering these objects on calling drawChildren() render them by layer order to pass OpenGL Z-Buffer Test.
